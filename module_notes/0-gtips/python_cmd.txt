1、
Python –m pip –-version   查看当前Python对应的pip版本；
Python –m pip list  	  查看当前Python对应的pip安装的第三方库；
Python –m pip install 	  库名即可安装对应的扩展库；
Python –m pip uninstall   库名即可卸载对应的扩展库
