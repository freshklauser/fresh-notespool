1. 参数C (C：default=1)
	软间隔引入的松弛因子的和的系数，惩罚系数 C*sum(松弛因子)
		当C趋近于无穷大时：意味着分类严格不能有错误 	  （C越大，松弛因子和就越小，对分类错误的容忍度就越小，越趋向 硬间隔）
		当C趋近于无穷小时：意味着可以有更大的错误分类容忍 （C越小，松弛因子和就越大，对分类错误的容忍度就越大，越趋向 软间隔）
		
2. kernel
	使用核函数的好处就是：可以在一个低纬空间完成高维空间的样本內积的计算。
		K(x,y) = ( <x, y> )**2 = <f(x), f(y)>
		          低纬內积    <--  高纬內积
